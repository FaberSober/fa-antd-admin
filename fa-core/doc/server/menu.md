# 菜单
## 菜单属性定义
1. 菜单ID格式定义为`aa.bb.cc.dd`(如10010203)
  1. aa=10，代表一个顶部模块
  2. bb=01，代表一个模块
  3. cc=02，代表这个模块下的一个菜单
  4. dd=03，代表这个菜单下的功能列表
