# 后台约定的一些规则
1. tree结构数据，根结点的ID默认为0；
2. enum类型的数据，在数据库中建议存储为tinyint(4)类型字段；

## 常用字段定义
| 字段名   | 数据类型         | 默认值  | 描述   |
|-------|--------------|------|------|
| id    | int(11)      |      | 主键   |
| sort  | int(11)      |      | 排序字段 |
| info  | varchar(225) |      | 描述   |
| valid | tinyint(4) | 是否有效 |
